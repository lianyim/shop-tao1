### 简介
    shop-tao是一个淘宝领取优惠券和分享日常的小程序

    目前已实现的功能：
        用户登录
        首页轮播
        疯抢排行榜
        9.9包邮
        换季收纳
        商品分类
        分类查询商品列表
        首页优质商品推荐
        商品搜索
        商品详情
        商品收藏
        口令购买
        分享日常
        点赞评论
        美团、饿了么领红包
        等等

### 核心技术
#### 依赖
- mysql5.7
- jdk1.8及以上
#### 前端
- uniapp
#### 后端
- springboot
- spring security
- redis
- mybatis
- jpa
- fastjson
